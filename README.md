# Orbital Atlas

A small interactive space-view globe built with Next.js, TypeScript, Tailwind CSS, and Three.js.

## Features

- Smooth drag / swipe rotation
- Gentle zoom in and out
- Auto-rotating globe with atmosphere and cloud layer
- Procedural stars and planet texture
- Vercel-ready Next.js App Router setup

## Scripts

```bash
npm install
npm run dev
npm run build
npm run start
```
